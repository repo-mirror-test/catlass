# Gemm Example Readme

├── 15_gemm

## 使用示例
- 获取代码之后编译相应的算子可执行文件，可参考[quickstart](../../docs/quickstart.md#算子编译)
- 执行算子
```
# cd [代码仓路径]/output/bin
# 可执行文件名 |矩阵m轴|n轴|k轴|Device ID
# Device ID可选，默认为0
./15_gemm 256 512 1024 0
```
执行结果如下，说明精度比对成功。
```
Compare success.
```
